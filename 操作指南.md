# 《星界旅者》v0.2 操作指南

> Web 版本 · Canvas 渲染 · 键鼠操作
> 运行地址：http://localhost:3000/

---

## 一、启动游戏

```
cd "d:\University\我的游戏demo\astral-wanderer"
npm run dev --cache ".npm-cache"
```

浏览器打开 `http://localhost:3000/`，即可看到主菜单。

## 三、角色选择

展示 4 位可选旅者，每位角色卡片显示：
- **名称** + **背景描述**
- **HP**（生命值）
- **盾**（初始护盾）
- **攻**（基础攻击力）
- **初始碎片**（开局携带的能力）

| 角色 | 特点 | 适合玩法 |
|------|------|---------|
| **焰心旅者** | HP 90 · 攻 12 · 初始碎片：灼热打击（火焰+燃烧） | 高攻脆皮，追求爆发 |
| **霜语旅者** | HP 80 · 盾 20 · 初始碎片：冰霜屏障（护盾+冻结） | 控制型坦克 |
| **影织旅者** | HP 70 · 攻 13 · 初始碎片：暗影步（闪避） | 高风险高回报 |
| **共鸣旅者** | HP 80 · 攻 10 · 初始碎片：虚空共鸣（击杀回血） | 续航型 |

| 操作 | 方式 |
|------|------|
| 选择角色 | 点击角色卡片上的 **「选 择」** 按钮 |
| 返回主菜单 | 按 **ESC** 键 |

## 四、地图探索（核心界面）

选择角色后自动进入地图探索状态。

### 4.1 界面分区

```
┌─────────────────────────────────────────────┐
│ 顶部状态栏 (Top Bar)                         │
│ HP条 · 护盾 · AP能量点 · 星尘 · 楼层/主题    │
├─────────────────────────────────────────────┤
│                                             │
│           楼层地图面板                        │
│   ┌──┐   固定房间  ← 当前房间(金色边框)       │
│   │S │─┐ 未探索房间显示 "?"                 │
│   └──┘ │ 连线 = 可通行路径                   │
│        │ 虚线 = 未知路径                     │
│   ┌──┐  ← 导航按钮（点击进入）               │
│   │? │                                      │
│   └──┘                                      │
├─────────────────────────────────────────────┤
│ 导航按钮栏                                    │
│ [→ 战斗] [→ 精英] [→ 商店] …               │
├─────────────────────────────────────────────┤
│ 提示文字：点击导航按钮进入房间 · ESC 返回     │
└─────────────────────────────────────────────┘
```

### 4.2 房间类型图标

| 图标 | 类型 | 说明 |
|------|------|------|
| **S** | 起始房 | 安全，无战斗 |
| **!** | 战斗房 | 1-2 个普通敌人 |
| **E** | 精英房 | 强力敌人，掉落更好 |
| **$** | 商店 | 花费星尘购买碎片/补给 |
| **&** | 祭坛 | 付出代价换取力量 |
| **R** | 休息点 | 恢复 30% HP |
| **T** | 宝箱房 | 免费获得碎片 |
| **B** | BOSS 房 | 楼层守护者，击败后进入下一层 |

### 4.3 操作方式

| 操作 | 方式 |
|------|------|
| 进入相邻房间 | 点击地图面板下方的 **导航按钮**（如 `→ 战斗`） |
| 未探索房间 | 导航按钮显示为 `→ ? 战斗`（带问号前缀） |
| 返回主菜单 | 按 **ESC** 键 |

### 4.4 地图结构

每层地图采用**固定房间位置**模板，房间类型**随机分配**：

- **第 1 层 · 尘封回廊**：8 个房间，分叉路径
- **第 2 层 · 异化深渊**：9 个房间，菱形结构
- **第 3 层 · 寂灭核心**：10 个房间，金字塔结构

起始房 → 多条可选路线 → 最终指向 BOSS 房。

## 五、回合制战斗

### 5.1 战斗界面

```
┌─────────────────────────────────────────────┐
│ 顶部状态栏                                   │
├─────────────────────────────────────────────┤
│            敌人面板                           │
│  [敌人图形] 敌人ID                           │
│  ████████░░░░  HP条                         │
│  ▓▓▓▓ 护盾                                   │
│  意图: 攻击 15 / 蓄力 30 / 防御              │
├─────────────────────────────────────────────┤
│  选择行动:                                   │
│  ┌──────────┐ ┌──────────┐                  │
│  │ 碎片名称  │ │ 碎片名称  │                 │
│  │ 2 AP     │ │ 1 AP     │                 │
│  └──────────┘ └──────────┘                  │
│                     [结 束 回 合]             │
├─────────────────────────────────────────────┤
│ 提示信息                                     │
└─────────────────────────────────────────────┘
```

### 5.2 AP 行动点系统（核心机制）

- 每回合获得 **3 点 AP**（● ● ●）
- 使用碎片消耗 1-3 AP
- **未使用的 AP 自动转化为护盾**（1 AP = 3 护盾）
- 护盾上限 30

### 5.3 敌人意图系统

每回合敌人会**提前预报**其行为：

| 意图 | 效果 | 应对策略 |
|------|------|---------|
| **攻击 N** | 造成 N 点伤害 | 本回合叠加护盾防御 |
| **蓄力 N** | 放弃本回合，下回合释放 N 伤害 | 优先打断（冻结/控制）或堆叠护盾硬扛 |
| **防御** | 为自己叠加护盾 | 使用物理伤害（无视 50% 护盾） |
| **治疗** | 回复自身 HP | 优先击杀目标 |
| **自爆 N** | 冲向玩家造成 N 伤害并死亡 | 残血时尽快击杀 |

### 5.4 操作方式

| 操作 | 方式 |
|------|------|
| 使用碎片攻击 | 点击碎片卡片（AP 足够时高亮，不足时灰色不可用） |
| 结束回合 | 点击 **「结 束 回 合」** 按钮 |
| 战斗胜利后 | 点击 **「继 续」** 返回探索 |
| 返回主菜单 | 按 **ESC** 键 |

## 六、状态效果

| 效果 | 持续 | 作用 |
|------|------|------|
| 🔥 燃烧 | 3 回合 | 每回合损失 5% 最大 HP |
| ❄️ 冻结 | 1 回合 | 无法行动 |
| 🐢 减速 | 2 回合 | AP 上限 -1 |
| ☠ 中毒 | 2 回合 | 每回合损失 8% 最大 HP |
| 🛡 护盾 | 永久(上限30) | 吸收伤害 |
| 💪 力量 | 2 回合 | 攻击伤害 +25% |
| 🔓 脆弱 | 2 回合 | 受到伤害 +25% |
| 🌫 闪避 | 1 回合 | 50% 概率躲避攻击 |

## 七、碎片系统

碎片是核心 Build 装备，分为 4 个稀有度：

| 稀有度 | 颜色 | 边框色 |
|--------|------|--------|
| 普通 | 灰色 | — |
| 稀有 | 蓝色 | `#5599dd` |
| 史诗 | 紫色 | `#aa55dd` |
| 传说 | 金色 | `#f0a030` |

### 碎片槽位

| 槽位 | 作用 |
|------|------|
| 主武器 | 核心攻击技能 |
| 副武器 | 辅助技能（控制/回复） |
| 护甲 | 生存能力（HP/护盾） |
| 饰品 1 | 被动增益 |
| 饰品 2 | 被动增益 |
| 共鸣器 | 系统改造（改变 AP 规则） |
